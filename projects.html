Conteneurisation d'une application web​​

Aout-Septembre 2022    -    ESIEA, Paris

Conteneurisation d'une application web appelée Udagram en utilisant Docker et déploiement de celle-ci sur un cluster Kubernetes composé de trois machines virtuelles.

Conteneurisation de l'application Udagram localement (Docker Desktop):
  • Clonage de l'application Udagram existante.
  • Création des fichiers Dockerfile pour chaque micro-service de l'application.
  • Configuration des fichiers .yaml pour définir les services, les déploiements et les volumes.
  • Test des fichiers de configuration pour s'assurer que les conteneurs fonctionnent correctement.
  • Création des images Docker pour chaque micro-service.
  • Publication (push) des images sur DockerHub pour pouvoir les récupérer ultérieurement.

Orchestration et déploiement de l'application avec Kubernetes sur des VMs Ubuntu :
  • Création de trois machines virtuelles, comprenant une machine maître (master) et deux machines esclaves (workers), pour former le cluster Kubernetes.
  • Installation des composants kubelet et kubeproxy sur les machines esclaves.
  • Jointure des machines esclaves au maître pour qu'elles puissent participer au cluster.
  • Création des fichiers de déploiement Kubernetes (.yaml) pour chaque micro-service de l'application, définissant les conteneurs, les réplicas, les volumes, etc.
  • Création des fichiers configmap et secrets pour la gestion des configurations et des données sensibles.
  • Création des fichiers de service Kubernetes (.yaml) pour exposer les services internes de l'application.
  • Configuration du port forwarding pour permettre l'accès externe au reverse proxy et à l'interface utilisateur (frontend) de l'application.

COMPETENCES & OUTILS

Conteneurisation, Orchestration, Docker, Kubernetes, Docker Hub, Visual Studio Code, VMWare, Git
